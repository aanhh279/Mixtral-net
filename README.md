# Overview

**Mixtral-net** is an experimental research project focused on studying several gameplay manipulation mechanisms in **Free Fire (Garena)**.

Current research includes:

* Teleport
* Ghost
* Freeze

The purpose of this repository is to explore software engineering concepts, reverse engineering, memory analysis, and related research topics. This project is not affiliated with or endorsed by Garena.

# Installation

```bash
git clone https://github.com/aanhh279/Mixtral-net.git
cd Mixtral-net
```

Open the project using **Android Studio**.

If optional AI features are enabled, create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Build and run the application.
